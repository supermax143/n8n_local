# n8n Local

Локальная установка n8n для разработки и тестирования.

## Установка

```bash
npm install
```

## Запуск

### Способ 1: Использование скриптов (рекомендуется)

**Windows (BAT):**
- Двойной клик по `start.bat` для запуска
- Двойной клик по `stop.bat` для остановки

**Windows (PowerShell):**
```powershell
.\start.ps1    # Запуск
.\stop.ps1     # Остановка
```

### Способ 2: Через npm

```bash
npm start
```

После запуска n8n будет доступен по адресу: http://localhost:5678

## Запуск с туннелем (для внешнего доступа)

```bash
npm run start:tunnel
```

## Остановка

- Используйте `stop.bat` или `stop.ps1`
- Или нажмите `Ctrl+C` в терминале, если запущено через `npm start`
